## Run locally
pip install -r requirements.txt
streamlit run app.py

## Deploy on Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Click **New app**.
3. Select this repo and set `app.py` as the main file.
4. Deploy!
